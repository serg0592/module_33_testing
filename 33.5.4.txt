тест-кейсы для возведения числа в квадрат

1) Передача на вход целого числа
    условие: передача на вход int числа
    шаги: передать на вход функции возведения в квадрат целые числа 2, -4, 10
    ожидание: 4, 16, 100

2) Передача на вход дробного числа
    условие: передача на вход float числа
    шаги: передать на вход функции возведения в квадрат дробные числа 0.2, -0.4, 0.01
    ожидание: 0.04, 0.016, 0.0001

3) Передача на вход строки
    условие: передать на вход функции возвежения в квадрат string строки
    шаги: передать на вход функции возведения в квадрат строки 'str', '2', '@#$%'
    ожидание: ошибка, возведение в квадрат невозможно, на вход должны поступать данные типа float и int

4) Передача на вход объекта
    условие: передать на вход объект типа object
    шаги: передать на вход функции возведения в квадрат объект testObject{}
    ожидание: ошибка, возведение в квадрат невозможно, на вход должны поступать данные типа float и int

5) Передача на вход массива
    условие: передать на вход массив array[]
    шаги: передать на вход функции массив testArr = []
    ожидание: ошибка, возведение в квадрат невозможно, на вход должны поступать данные типа float и int
